# Contributing to ruview

Thank you for your interest in contributing!

## Project Overview

**ruview** — π RuView turns commodity WiFi signals into real-time spatial intelligence, vital sign monitoring, and presence detection — all without a single pixel of video.

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes and commit them with clear, descriptive messages
4. Push to your fork: `git push origin feature/your-feature-name`
5. Open a Pull Request against the `main` branch

## Pull Request Process

- Ensure your code follows the existing style and conventions
- Update documentation as needed
- The PR should pass all existing tests and linting checks
- At least one maintainer review is required before merging

## Code of Conduct

Please be respectful and constructive in all interactions. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).
